2. Redaction MUST NOT short-circuit on `typeof value !== 'object'` for top-level strings â€” a tool input passed as a single string is the most common form of secret leakage in MCP tool events.
3. Every redactor MUST recurse into arrays AND objects with the same rule set; a redactor that handles `Array.isArray` but stops at `Object.entries` will miss any nested array of objects (e.g. message-history payloads).
